Az 1 és 5 közötti árú termékek lekérdezése:

```
select * from Aruk where ar > 1 and ar < 5
```

```
select * from Aruk where ar between 1 and 5
```

1. feladat: Kérdezzük le néhány termék adatai az ARUK adattáblából. Konkrétan: az eredménytáblában szerepeljenek az 1-nél kisebb árú termékek ÉS az 5-nél nagyobb áru termékek

```
select * from Aruk where ar < 1 or ar > 5
```

```
select * from Aruk where ar < 1
union
select * from Aruk where ar > 5
```

Tranzakciok és Vasarlok tábla sorainak párosítása:

```
select * from Tranzakciok

select * from Vasarlok where vasarlo_id = "V02"

select *
from Tranzakciok, Vasarlok
where vevo = vasarlo_id

select *
from Tranzakciok full outer join Vasarlok
on vevo = vasarlo_id
/* Ez MySQL-ben valamiert nem mukodott...*/


select * from Tranzakciok left outer join Vasarlok
on vevo = vasarlo_id
union 
select * from Tranzakciok right outer join Vasarlok
on vevo = vasarlo_id
```

2. feladat: Vegyük hozzá a Tranzakciok adattáblához a termékek adatait: 

```
select * from (Tranzakciok join Aruk on termek = cikkszam) 
left outer join Vasarlok on vevo = vasarlo_id;
```

3. feladat: Vegyük hozzá a Tranzakciok adattáblához a termékek és vásárlók adatait:

```
select * from Tranzakciok, Aruk, Vasarlok
where termek = cikkszam and vevo = vasarlo_id

select * from 
(Tranzakciok left outer JOIN Aruk on termek = cikkszam)
left outer join Vasarlok on vevo = vasarlo_id
```

4. felatat: Bevétel számolása napi szintű lebontásban
   
```
select datum, sum(round(ar*mennyiseg,2))
from Tranzakciok join Aruk on termek = cikkszam
group by datum
```
