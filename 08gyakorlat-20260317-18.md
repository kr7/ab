Az ARUK tábla AR oszlopának típusát megváltoztatjuk. A DECIMAL(10,2) legfeljebb 10 jegyű számokat tárol, amelyek közül 2 jegy áll a tizedesvessző után:

```
alter table Aruk modify column Ar decimal(10, 2);
```

Mennyi pénzt költött az ügyfél a T0001-es azonosítószámú
   tranzakcióban (vásárlás során)?

```
select tranzakcio_id, sum(mennyiseg*ar) as teljes_ar 
from Tranzakciok, Aruk
where tranzakcio_id = 'T0001' and 
Tranzakciok.termek = Aruk.cikkszam;
```

Mennyi pénzt költöttek az ügyfelek az egyes tranzakciókban (azaz tranzakciónként)?

```
select tranzakcio_id, sum(mennyiseg*ar) 
from Tranzakciok, Aruk
where Tranzakciok.termek = Aruk.cikkszam
group by tranzakcio_id
```

```
select * from 
	(select tranzakcio_id, sum(mennyiseg*ar) as ertek
	from Tranzakciok, Aruk
	where Tranzakciok.termek = Aruk.cikkszam
	group by tranzakcio_id) as reszeredmeny
where ertek > 100
```

Határozzuk meg minden ügyfélre, hogy hány tranzakcióban
érintettek, az eredménytáblában az ügyfeleket a nevükkel
jelenítsük meg.

```
select nev, tranzakciok_szama
from Vasarlok right join 
	(select vevo, count(*) as tranzakciok_szama
	from (select distinct tranzakcio_id, vevo from Tranzakciok) 
		 as tranzakcio_id_vevo
	group by vevo) as tranzakciok_szama_vevokoddal
on Vasarlok.vasarlo_id = tranzakciok_szama_vevokoddal.vevo;
```

```
create view TranzakciokSzamaUgyfelenkent as 
(select nev, tranzakciok_szama
from Vasarlok right join 
	(select vevo, count(*) as tranzakciok_szama
	from (select distinct tranzakcio_id, vevo from Tranzakciok) 
		 as tranzakcio_id_vevo
	group by vevo) as tranzakciok_szama_vevokoddal
on Vasarlok.vasarlo_id = tranzakciok_szama_vevokoddal.vevo);
```

```
select * from TranzakciokSzamaUgyfelenkent;
```

áru termékeket tartalmazza. 

```
create view OlcsoAruk as
(select * from Aruk where ar < 1)
```

```
select cikkszam, megnevezes from OlcsoAruk
```
